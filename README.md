# Zara Sales Analysis Dashboard

Interactive sales analysis dashboard created using **SQL Server** and **Power BI**.  
This end-to-end data analysis project focuses on sales performance, product ranking, seasonality impact, promotion analysis, and revenue segmentation for an e-commerce fashion dataset inspired by Zara products.

---

## Project Overview

The goal of this project was to analyze e-commerce sales data and build an interactive business intelligence dashboard supporting decision-making.

The analysis focuses on:
- identifying top-performing products
- analyzing the impact of promotions on revenue
- comparing seasonal vs non-seasonal products
- analyzing revenue distribution across product categories and sections
- product positioning insights

The project covers the full analytical workflow from raw data preprocessing to dashboard visualization.

## Project Workflow

### 1. Data Preparation in SQL Server
The dataset was imported into SQL Server and prepared for analysis.

Data preparation included:
- importing CSV data into SQL Server
- cleaning and preprocessing data
- correcting data types
- renaming and formatting columns
- preparing analytical queries

---

### 2. SQL Analysis & Views
Analytical SQL views were created to structure business insights for Power BI reporting.

Created views include:
- total revenue
- average product price
- top 5 products by revenue
- product ranking
- revenue by product category
- revenue by section
- promotion impact analysis
- seasonal impact analysis
- product position analysis

SQL techniques used:
- aggregations
- GROUP BY
- window functions (`RANK()`)
- filtering
- analytical views

---

### 3. Power BI Dashboard Development
SQL views were imported into Power BI to create an interactive dashboard.

Dashboard features:
- KPI cards
- product ranking table
- Top 5 products revenue visualization
- revenue by product type
- promotion analysis
- seasonal analysis
- revenue by section
- interactive slicers and filtering
